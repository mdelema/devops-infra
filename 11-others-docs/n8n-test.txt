	####### 	Comandos de PM2  	########## 
	> pm2 stop n8n -------------------> Detiene el servicio
	> pm2 delete n8n -----------------> Elimina lo que preconfigurado en n8n
	> pm2 start ecosystem.config.js --> Levanta n8n con los parametros del config.js
	> pm2 reload ecosystem.config.js -> Reinicia y Aplica los cambios
	> pm2 save -----------------------> Guarda la configuración. 
	> pm2 show n8n -------------------> Te da un detalle del proceso y las metricas
	> pm2 list -----------------------> Lista los procesos que estan corriendo
	> pm2 env 0 ----------------------> Ver todas las variables de entorno que PM2 está pasando al proceso n8n (id 0).
	> pm2 startup --------------------> Detect que usás systemd y crea el servicio /etc/systemd/system/pm2-root.service.
	> ps aux | grep n8n --------------> para ver si esta corriendo
